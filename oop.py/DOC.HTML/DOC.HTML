Introduction

Object-oriented programming (OOP) is a programming paradigm based on the concept of "objects", which can contain data and code: data in the form of fields (often known as attributes or properties), and code, in the form of procedures (often known as methods).

For more information, visit Example.

Classes

A class is a blueprint for creating objects. It defines a set of attributes and methods that the created objects will have.

Objects

An object is an instance of a class. It contains data and methods that operate on that data.

Inheritance

Inheritance allows a class to inherit attributes and methods from another class.

Polymorphism


Polymorphism allows for methods to do different things based on the object it is acting upon.
